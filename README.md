# NotifyX Backend

A backend system built with Node.js, Express, TypeScript, and MongoDB providing authentication, role-based access (planned), and scalable API architecture.

- Node.js
- Express.js
- TypeScript
- MongoDB + Mongoose
- JWT Authentication
- Zod validation

- ### Current Features
  
- User registration
- User login
- JWT authentication
- Global error handling
- Request validation using Zod

### Upcoming Features
- Auth middleware protection
- Role-based access control
- Notifications system
